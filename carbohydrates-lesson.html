Задание 5
🍳 «Мой завтрак для учёбы»
Составь завтрак из 3–4 продуктов перед сложным экзаменом (4 часа). Система проанализирует твой выбор по типу углеводов.

Конструктор завтрака перед экзаменом

Нужна стабильная энергия без резких скачков сахара

Условие: завтра экзамен — 4 часа концентрации. Мозгу нужна стабильная глюкоза. Простые углеводы дадут быстрый подъём и резкий спад. Составь завтрак, который продержит тебя 4 часа.

Анализ завтрака
Пример идеального завтрака: Овсяная каша (сложные углеводы → долгая энергия) + яйцо (белок → сытость) + горсть орехов (жиры → замедляют всасывание) + зелёный чай без сахара. Избегай: сладкие хлопья, белый хлеб с джемом, сладкий йогурт — быстрые углеводы вызовут «сахарные качели».
